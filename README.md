This library is the generator for Karee projects

## Usage

A simple usage is:

- Add dependency
```yaml
    karee_injectable_gen: ^1.0.1
```

- Install Karee-CLI
  
  Github Repo:  [Karee-CLI](https://github.com/ChamplainLeCode/karee-cli.git)
```bash
    npm install -g karee
```

- Run the CLI command to generate sources

```bash
    karee source
```
